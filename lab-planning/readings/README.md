# Reading Digests

One digest per reading. Purpose: compressed reference for the TA who hasn't
re-read everything. Not course-facing.

## Structure

- `intro/` — Tiqqun, McQuillan.
- `week-01/` through `week-13/` — primary readings per week, plus
  Additional/Primary Documents assigned for that week.
- `uncategorized/` — files present in `markdown/` that aren't in any
  week's reading list but are in the repo (Malabou, Doctorow Chickenized,
  Brusseau, Yuk Hui). Flagged for verification with Scott.

## Digest format

Every file uses this template:

```
# [Author], [Title]

**Week:** [N | Intro | Additional-WN | Uncategorized]
**File:** `markdown/<path>`
**Length:** N lines

## Summary
2–4 sentences, plain language.

## Key quotes
- "…" — location (line N or p. X).
- …

## Why it's in the course
1–2 sentences on how it connects to that week's seminar topic.

## Relevance to the lab
1–2 sentences on how (or whether) it informs the four-stance lab frame.
```
